Daniel Eno's - HTML Cheat Sheet

This HTML cheat sheet provides a quick reference to common HTML tags and their usage.

Welcome to my HTML cheat sheet. Use this guide to learn and reference HTML tags.

Tag Description Example
<html> Defines the root of an HTML document. <html>...</html>
<head> Contains meta-information about the document. <head>...</head>
<title> Sets the title of the document (shown in the browser tab). <title>Page Title</title>
<body> Contains the visible content of the document. <body>...</body>
<h1> to <h6> Defines header text with different levels of importance. <h1>Header 1</h1>
<p> Represents a paragraph of text. <p>This is a paragraph.</p>
<a> Creates a hyperlink. <a href="https://example.com">Visit Example</a>
<img> Embeds an image in the document. <img src="image.jpg" alt="Description" width="100" height="80">
<ul> and <li> Defines an unordered list and list items. <ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
<ol> and <li> Defines an ordered list and list items. <ol>
<li>Item 1</li>
<li>Item 2</li>
</ol>
<table>, <tr>, <td>, <th> Creates a table, table rows, table data cells, and table headers. <table>
<tr>
<th>Header 1</th>
<th>Header 2</th>
</tr>
<tr>
<td>Row 1, Cell 1</td>
<td>Row 1, Cell 2</td>
</tr>
</table>
<a> Attribute: href Specifies the URL of the linked resource. <a href="https://example.com">Visit Example</a>
<img> Attributes: src, alt, width, height Defines the image source, alternative text, width, and height. <img src="image.jpg" alt="Description" width="100" height="80">
<ul> and <li> Defines an unordered list and list items. <ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
<ol> and <li> Defines an ordered list and list items. <ol>
<li>Item 1</li>
<li>Item 2</li>
</ol>
<table>, <tr>, <td>, <th> Creates a table, table rows, table data cells, and table headers. <table>
<tr>
<th>Header 1</th>
<th>Header 2</th>
</tr>
<tr>
<td>Row 1, Cell 1</td>
<td>Row 1, Cell 2</td>
</tr>
</table>
<em> Represents emphasized text, typically displayed in italics. <em>This text is emphasized.</em>
<strong> Represents strong importance, typically displayed in bold. <strong>This text is strong.</strong>
<br> Inserts a line break within text content. This is some text.<br>This is on a new line.
<blockquote> Defines a section that is a quotation from another source. <blockquote>
This is a quotation.
</blockquote>
